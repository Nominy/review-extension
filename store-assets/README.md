# Store Assets

This directory holds Chrome Web Store submission assets that are intentionally kept outside the extension package.

Generated files:

- `screenshots/release-options.png`
- `screenshots/release-summary.png`
- `promo/small-promo-tile.png`
- `promo/marquee-promo-tile.png`

These files are for store submission and should not be copied into the packaged extension zip.
