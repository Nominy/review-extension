# Chrome Web Store Listing Copy

## Name

Babel Review Helper

## Short Description

Guided review helper for Babel transcription reviewers with fast apply and interactive session workflows.

## Detailed Description

Babel Review Helper helps reviewers work inside the Babel dashboard by capturing the current review state, generating structured feedback, and applying comments back into the review form.

Features:

- one-click Magic Review entry point
- fast workflow for direct feedback application
- interactive review sessions with per-change inspection
- manual template matching and reviewer comments
- production-only Chrome Web Store build with minimal permissions

## Support Notes

- Intended for Babel reviewers working on `https://dashboard.babel.audio/*`.
- Requires access to the production review backend.
- Chrome Web Store release build does not use localhost backends.
